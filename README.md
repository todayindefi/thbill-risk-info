# thBILL Risk Report

Independent risk assessment dashboard for Theo Network's thBILL (Short Duration US Treasury Fund).

## Live Dashboard

[View Dashboard](https://your-username.github.io/thbill-risk-info/)

## Features

- **Live Metrics**: TVL, backing ratio, 24h flow
- **Collateral Analysis**: ULTRA supply across Ethereum, Arbitrum, Solana
- **Treasury Holdings**: On-chain verified ULTRA holdings
- **Secondary Liquidity**: DEX pools and 24h volume
- **DeFi Integrations**: Pendle, Morpho, Euler, and more
- **Risk Analysis**: Issuer, decentralization, technical, operational risks

## Data Sources

- On-chain RPC queries (Ethereum, Arbitrum, Solana)
- [DefiLlama](https://defillama.com/protocol/theo-network-thbill) - TVL and yields
- [CoinGecko](https://www.coingecko.com/en/coins/theo-short-duration-us-treasury-fund) - DEX tickers

## Architecture

```
thbill-risk-info/
├── index.html          # Main dashboard
├── js/app.js           # Data fetching and UI updates
├── css/custom.css      # Custom styles
├── data/               # Metrics JSON (synced from PegTracker)
└── .github/workflows/  # GitHub Pages deployment
```
